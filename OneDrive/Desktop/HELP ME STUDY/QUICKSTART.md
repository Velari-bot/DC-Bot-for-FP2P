Get your AI Study App running in 3 simple steps!

## Step 1: Install Dependencies

Open your terminal in this directory and run:

```bash
npm install
```

This will install all the required packages (React, TypeScript, Vite, TailwindCSS, Zustand).

## Step 2: Start the Development Server

```bash
npm run dev
```

Wait a few seconds for the server to start.

## Step 3: Open in Browser

The terminal will show you a URL (typically `http://localhost:5173`). Click it or copy-paste into your browser.

## What to Try First

1. **Multiple Choice Quiz** - Start here to get familiar with the questions
2. **Flashcards** - Great for memorization and quick review
3. **Speed Mode** - Challenge yourself with a 60-second timer
4. **Type Your Answer** - Test your recall without hints
5. **Matching Game** - Fun visual way to connect questions and answers
6. **Weak Spot Drill** - Will unlock after you've answered some questions incorrectly

## Your Progress is Saved!

Everything is automatically saved to your browser's local storage:
- Which questions you've seen
- How many times you got each question right/wrong
- Your best streak
- Overall accuracy

You can close the browser and come back anytime - your progress will be preserved!
