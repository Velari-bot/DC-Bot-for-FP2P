# Stripe Webhook for Vercel

This webhook handler matches the Firebase Functions implementation and integrates with Firestore.

## Features

- ✅ Full Firestore integration
- ✅ Subscription management
- ✅ Usage limit resets
- ✅ Premium activation/cancellation
- ✅ All Stripe events handled

## Setup

1. Install dependencies: `npm install firebase-admin`
2. Set environment variables in Vercel
3. Update Stripe webhook URL to point to Vercel
4. Deploy

See `UPDATE-STRIPE-WEBHOOK-TO-VERCEL.md` for full instructions.
